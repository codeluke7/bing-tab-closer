# Bing Tab Closer

One-click cleanup of all open Bing tabs.

Click the extension icon and every bing.com tab disappears instantly.

No tracking.
No analytics.
No nonsense.

# Features

Closes all open Bing tabs with one click

Lightweight, minimal permissions

No background tracking

Fully open source

# How It Works

When the toolbar button is clicked, the extension:

Searches for tabs matching `*://*.bing.com/*`

Collects their tab IDs

Closes them

That’s it.

# Permissions

This extension only requests:

tabs – Required to find and close Bing tabs.

It does not:

Collect data

Send network requests

Store user information

Inject scripts into websites

# Installation (Manual)

Download or clone this repository.

Go to chrome://extensions

Enable Developer mode

Click Load unpacked

Select the Bing Tab Closer folder.

# Installation automatic

Go to releases

Download ZIP file

Unzip the file

Follow steps above 

*A CRX file is included in the release but it is not recommended to use it as it may cause errors.*
